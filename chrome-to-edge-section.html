Switching Browsers

Moving from Chrome to Edge?
Your bookmarks come with you.

You don't need to start from scratch. Edge can bring across your bookmarks and saved sites so your day-to-day browsing carries on without missing a beat.

No need to relearn everything

If you mainly use Chrome out of habit, the switch to Edge is straightforward. The goal is simply to make sure your important sites and day-to-day browsing still work smoothly — not to change how you work.

How to import
Two ways to bring your bookmarks across
Option 1 — Quickest

Import directly from Chrome

If Chrome is still installed on your device, Edge can pull your bookmarks, saved passwords, and browser data across in just a few clicks — no file needed.

  • 1 Open Edge and go to Settings → Profiles → Import browser data
  • 2 Choose Google Chrome as the source
  • 3 Select what you want to import and confirm
Option 2 — File-based

Import from a bookmarks file

If you've already exported your Chrome bookmarks as an HTML file — or Chrome isn't on this device — Edge can import directly from that file.

  • 1 Open Edge and go to Settings → Profiles → Import browser data
  • 2 Choose Favourites or bookmarks HTML file
  • 3 Browse to your saved file and import

Where do imported bookmarks appear?

Once imported, your favourites will show up in a folder called Imported on the Edge favourites bar. From there you can organise them however suits you — move them into folders, pin the ones you use most, or just leave them as-is.
